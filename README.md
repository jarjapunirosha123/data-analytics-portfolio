# HR Analytics Dashboard – Power BI

## Project Overview
This project presents an HR analytics dashboard built using Power BI to analyse workforce data and identify key employee trends.

## Tools Used
Power BI  
Excel  
Data Visualization

## Dashboard Features
- Employee headcount overview
- Active vs terminated employees
- Gender distribution analysis
- Employee performance analysis
- Recruitment source insights
- Employee satisfaction analysis
- Department level workforce distribution

## Key Insights
- Most employees belong to the Production department
- Majority of employees fully meet performance expectations
- Recruitment sources like Indeed and LinkedIn contribute most hires
- Employee satisfaction levels are mostly high

## Purpose
The goal of this dashboard is to help HR teams analyse workforce metrics and support better decision-making using data visualisation.
